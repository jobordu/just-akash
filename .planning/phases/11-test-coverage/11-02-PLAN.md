`main()` function — implement these 6 steps in order, with all cleanup in a `try/finally`-style structure (use a top-level try block; the finally-equivalent is "always run destroy if dseq is set"):

**Step 1 — Validate environment (exit immediately if missing, no dseq needed yet):**
```python
for var in ("AKASH_API_KEY", "AKASH_PROVIDERS", "SSH_PUBKEY"):
    if not os.environ.get(var):
        log_fail(f"Required env var {var} is not set")
        sys.exit(1)
```

**Step 2 — Deploy via `just up` (timeout=300s):**
- Run `just up`
- On non-zero return code: log_fail, sys.exit(1) (no dseq yet, no cleanup needed)
- Extract DSEQ with `re.search(r"DSEQ[:\s]+(\d+)", r.stdout + r.stderr)` (proven pattern from test_lifecycle.py)
- If no DSEQ found: log_fail("Could not parse DSEQ from `just up` output"), sys.exit(1)
- log_pass(f"Deployed DSEQ={dseq}")

**Step 3 — Wait for lease readiness (poll up to 5×5s after initial 10s wait):**
```python
time.sleep(10)
lease_ready = False
for attempt in range(1, 6):
    r = run(f"uv run just-akash status --dseq {dseq}", timeout=30)
    if "hostUri" in r.stdout or "host_uri" in r.stdout:
        lease_ready = True
        break
    if attempt < 5:
        time.sleep(5)
if not lease_ready:
    failures.append("lease_timeout")
    log_fail("Lease not active after 35 seconds")
```

**Step 4 — exec via lease-shell (only if no failures yet):**
```python
r = run(f"uv run just-akash exec 'echo hello from lease-shell' --dseq {dseq}", timeout=30)
if r.returncode == 0 and "hello from lease-shell" in r.stdout:
    log_pass("exec: output verified")
else:
    log_fail(f"exec failed (rc={r.returncode}):\n{r.stderr}")
    failures.append("exec_failed")
```

**Step 5 — inject via lease-shell + verify via exec:**
- Write a temp file with content `TEST_SECRET=injected_value\n` using `tempfile.NamedTemporaryFile(mode='w', suffix='.env', delete=False)`
- Run `uv run just-akash inject --env-file {env_file} --dseq {dseq}` (timeout=30)
- Clean up temp file in a `finally` block
- On success, run `uv run just-akash exec 'cat /tmp/test.env' --dseq {dseq}` to verify
- Check `"injected_value" in r.stdout`
- Append "inject_failed" or "inject_verify_failed" to failures as appropriate

**Step 6 — Cleanup (always runs if dseq is set, even on earlier failure):**

Structure the entire Steps 3-5 block in a `try` with `finally` that runs `just destroy {dseq}`:
```python
try:
    # steps 3-5 here
except Exception as e:
    log_fail(f"Unexpected error: {e}")
    failures.append(str(e))
finally:
    if dseq:
        log_step(TOTAL_STEPS, f"Cleanup: destroy DSEQ={dseq}")
        r = run(f"just destroy {dseq}", timeout=60)
        if r.returncode == 0:
            log_pass("Destroyed")
        else:
            log_fail(f"destroy failed:\n{r.stderr}")
            failures.append("destroy_failed")
```

**Exit logic:**
```python
if failures:
    log_fail(f"{len(failures)} step(s) failed: {failures}")
    sys.exit(1)
else:
    log_pass("All steps passed — lease-shell transport validated end-to-end")
```

**`if __name__ == "__main__":` block at the bottom:**
